# Esys

##Â Prerequisites
- Node JS
```cli
npm install -g gulp
```

## Installation
```cli
git clone git@github.com:hamishhossack/esys.git
cd esys
npm install && bower install
```

## Run the app
> **NOTE:** We should be able to to get the client on http://localhost:9000/ 

```cli
gulp serve
```

## Run a build
```cli
gulp build
```

## Run tests
> **NOTE**: Add --client or --server to individually test

```cli
gulp test
```
