# Esys

## Installation
```cli
git clone git@github.com:hamishhossack/esys.git
npm install && bower install
```

## Run the app
> *NOTE:* We should be able to to get the client on http://localhost:9000/ 

```cli
gulp serve
```

## Run a build
```cli
gulp build
```

## Run tests
```cli
gulp test
```
