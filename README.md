# Esys

## Installation
```cli
git clone git@github.com:hamishhossack/esys.git
npm install && bower install
```

## Run the app

```cli
gulp serve
```

## Run a build
```cli
gulp build
```

## Run tests
> *NOTE:* The two API tests are failing due to an issue on connection to the api server.. still debugging. 

```cli
gulp test
```
